# Van Tharp's Marble Game

This project is a simulator for the Van Tharp Marble Game designed to illustrate the effect of sizing strategies.

- 65% black: 0
- 15% blue: 1
- 7% green: 2
- 6% yellow: 3
- 3% silver: 5
- 2% gold: 10
- 2% pink: 20

The game is described in the book [Trading Beyond the Matrix](https://www.amazon.com/Trading-Beyond-Matrix-Traders-Investors/dp/1118525663) by Van K. Tharp.

The web page of this project are just plain HTML, CSS, and Javascript (without external dependencies).

The code for Van-Tharp-Marble-Game is Open Source at [GitHub](https://github.com/evoluteur/van-tharp-marble-game) with MIT license.
